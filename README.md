# Standards-HRC Paperclip Skills

Skill bundles consumed by the Paperclip AI agent platform for **Standards Consultants (Standards-HRC)**.

## Skills in this repo

| Skill | Purpose |
|---|---|
| [`standards-hrc/`](./standards-hrc) | Company knowledge base — identity, services, clients, voice |
| [`design-guidelines/`](./design-guidelines) | Visual identity and presentation/document format playbook |
| [`operating-rules/`](./operating-rules) | Universal operating procedures (closing-the-loop, escalation, attribution) |

## Importing into Paperclip

In the Paperclip dashboard → **Skills** → paste the GitHub URL into the *"Paste path, GitHub URL, or skills.sh command"* field at the top, then click **Add**. Repeat for each of the three skill folders.

After import, attach each skill to the agents that need it via **Agents → [Agent name] → Skills** tab.

## Editing

Edit the `SKILL.md` files directly, commit, and push. Re-import in Paperclip to pick up the latest version (or use Paperclip's refresh-from-source if available).

## Source of truth

Visual identity and copy patterns derive from the **Standards Corporate Profile 2026 (15 Years of Excellence)**. When facts conflict between sources, the corporate profile wins.
